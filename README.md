\# Task Manager App



A Flutter-based task manager app built for an internship assessment.



\## Features



\### Required Features

\- ✅ \*Task List Screen\* – View all tasks with title, description, status, and date

\- ✅ \*Add Task\* – Create new tasks with title, description, and due date

\- ✅ \*Edit Task\* – Modify existing tasks

\- ✅ \*Delete Task\* – Remove tasks with confirmation dialog

\- ✅ \*Mark as Complete\* – Checkbox with visual indicator (strikethrough text)



\### Bonus Features Implemented

\- ✅ \*Dark Mode\* – Toggle between light and dark themes

\- ✅ \*Search Tasks\* – Filter tasks by title or description

\- ✅ \*Local Storage\* – Tasks persist after app restart using SharedPreferences
